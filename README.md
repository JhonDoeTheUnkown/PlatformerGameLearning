# 🧗‍♂️ 2D Platformer Adventure

Welcome to my 2D Platformer game repository! This project is currently under active development using Unity and C# as part of my advanced game development learning journey at **Coding Studio**. The goal is to build a polished, responsive, and fun side-scrolling platformer from scratch.

## 🚀 Key Features (Planned & In Development)
* **Precise Physics Movement:** Smooth horizontal running, acceleration, and crisp jumping mechanics using Unity's 2D Rigidbody.
* **Level Design & Tilemaps:** Structured world design using Unity's Tilemap system (Platforms, Hazards, and Backgrounds).
* **Collectible & Scoring System:** In-game items (e.g., coins/gems) that players can collect to increase their score.
* **Enemy AI & Hazards:** Basic patrolling enemies, spikes, and pits that trigger a player death/respawn sequence.
* **State Machine Animation:** Seamless transitions between Idle, Run, Jump, and Fall animations based on player velocity.

## 📂 Repository Structure

* **`Assets/Scripts/`**: C# scripts handling player state, physics, combat, enemy AI, and game managers.
* **`Assets/Sprites/`**: Visual assets including environment tileset sheets, UI, and character sprite sheets.
* **`Assets/Animations/`**: Animation controllers and clips for characters and interactive objects.
